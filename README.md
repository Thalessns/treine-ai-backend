<h2>Como rodar a aplicação: </h2>

1 - Instale o `Python >= v3.11` <br>
2 - Crie uma env para instalar as libs, ex: `python -m venv caminho_desejado/nome_da_venv` <br>
3 - Ative a env, ex: `. caminho_desejado/nome_da_venv/Scripts/activate` <br>
4 - Instale as libs, ex: `pip install -r requirements.txt` (Supondo que seu terminal esteja aberto na raiz do projeto) <br>
5 - Coloque as informações necessárias nas variáveis de ambiente, veja o arquivo `.env.examples` e crie um arquivo `.env` na raiz do projeto <br>
6 - Rode a aplicação, ex: `python entrypoint.py` (Supondo que seu terminal esteja aberto na raiz do projeto) <br>
